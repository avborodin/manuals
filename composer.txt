
====== openserver =================

Первым делом для установки компосера перейдем в папку с актуальной (которая используется в данный момент) версией php. В моем примере это версия 7.3.

cd modules\php\PHP_7.3

Вторым шагом загрузим необходимые файлы

php -r "readfile('https://getcomposer.org/installer');" | php

После успешного скачивания, проверим работоспособность компосера (проверив текущую его версию)

php composer.phar -V

Далее для удобства и того что-бы дальше не писать php composer.phar создадим батник

echo @php "%~dp0composer.phar" %*>composer.bat

Теперь (например) проверить ту-же версии можно простой командой

composer -V

====== openserver end =================
